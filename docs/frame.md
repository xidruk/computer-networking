# Frame

This document explains the concept of a frame in computer networking.

## What is a Frame?
A frame is a digital data transmission unit in computer networking and telecommunication. Frames are used at the Data Link Layer (Layer 2) of the OSI Model.

## Key Points
- Contains header, payload, and trailer
- Used in Layer 2 (Data Link Layer) of the OSI Model
- Responsible for error detection and flow control

## Examples
- Ethernet frames
- Data sent over local networks
